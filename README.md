# MidLauncher — Инструкция по сборке

## Что получится на выходе

| Платформа | Файл | Описание |
|-----------|------|----------|
| Windows | `MidLauncher-Setup-1.0.0.exe` | Установщик с визардом (путь, ярлыки, автозапуск) |
| Windows | `MidLauncher-1.0.0-Portable.exe` | Портативный .exe — ничего не устанавливает |
| Linux | `MidLauncher-1.0.0.AppImage` | Универсальный Linux, запускается без установки |
| Linux | `midlauncher_1.0.0_amd64.deb` | Пакет для Debian / Ubuntu |
| macOS | `MidLauncher-1.0.0.dmg` | Drag-and-drop установщик |

---

## Шаг 1 — Установи Node.js

Скачай и установи **Node.js LTS** с официального сайта:
👉 https://nodejs.org

Проверь что установилось — открой терминал (или cmd) и введи:
```
node --version
```
Должно показать что-то вроде `v22.x.x`. Если показало — всё готово.

---

## Шаг 2 — Распакуй проект

Распакуй скачанный zip в любую папку, например:
- Windows: `C:\MidLauncher-build\`
- Linux/macOS: `~/MidLauncher-build/`

Убедись что в папке есть файлы: `main.js`, `package.json`, `BUILD.bat` и т.д.

---

## Шаг 3 — Запусти сборку

### На Windows:
Двойной клик по файлу **`BUILD.bat`**

Откроется чёрное окно, начнётся скачивание зависимостей (~5 минут при первом запуске).
По завершению автоматически откроется папка `dist\` с готовыми файлами.

### На Linux:
```bash
chmod +x BUILD.sh
./BUILD.sh
```

### На macOS:
```bash
chmod +x BUILD.sh
./BUILD.sh
```

---

## Готово! Папка `dist\`

После сборки открой папку `dist\` — там будут готовые установщики.
